# n8n 本地部署指南

n8n 是一个强大的工作流自动化平台，允许您连接各种服务和应用程序，创建自动化工作流程，无需编写代码。本指南将帮助您在本地环境中部署 n8n。

## Install

### 1. 使用 npm 安装

1. 打开命令提示符，全局安装 n8n：

```bash
npm install n8n
```

2. 启动 n8n：

```bash
n8n start
```

启动后，在浏览器中访问 localhost:5678 即可使用 n8n。
